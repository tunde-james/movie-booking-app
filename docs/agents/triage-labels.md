# Triage Labels

Use these canonical states:

- `needs-triage`: unclear or not yet reviewed
- `needs-info`: blocked by missing human/product decision
- `ready-for-agent`: agent can implement with little or no extra context
- `ready-for-human`: human should implement or review
- `wontfix`: intentionally not planned

For this project, prefer `ready-for-human` by default because the developer wants to write the code.
